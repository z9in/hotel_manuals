고객 응대 방법

[기본 멘트]

1. (프로트로 고객이 전화하여 소통 시) 프론트입니다. 무엇을 도와드릴까요?
2. (양방향 CCTV로 키오스크 통하여 소통 시) 실례합니다. 도와드릴까요?
3. (프론트 고객용 전화기(070-4153-3979) 통하여 소통 시) 실례합니다. 도와드릴까요?

[민원고객 발생 시 대처 방안]

1. 객실 불만의 경우 (남은 객실 확인 후) 같은 등급의 객실이나 업그레이드된 등급의 객실로 변경
-> 남은 객실이 없을 경우 하단의 2, 3 방안 제시
2. 체크아웃 시간 최대 3시간까지 연장
3. 체크아웃 시 음료 제공
★ 야간 관제 시간에는 고객을 대면하여 상대하기 어려워 민원이 커질 가능성이 높기 때문에 민원이 커지지 않도록 항상 친절하게 대응을 원칙으로 함.
★ 야간 관제 후 민원 및 특이사항은 '일일관제내역'에 정확하게 기재하여 아르코 호텔로 공유될 수 있도록 함.

[고객 양해 응대 멘트]

1. (고객 불편함 호소 시) 불편드려 죄송합니다. 확인해보겠습니다.
2. (내용 확인 후) 기다려 주셔서 감사합니다. ~~~ 내용 안내
3. (고객이 오래 기다릴 경우) 오래 기다려 주셔서 감사합니다. ~~~ 내용 안내
4. (통화 종료 후 확인이 필요한 경우) 정확하게 확인 후 재연락 드리겠습니다. 바로 도움을 드리지 못하여 죄송합니다.
5. (응대 종료 시) 다시 한 번 불편드려 죄송합니다.
